# JoyTracker - Трекер Счастья

## Дата создания: 2026-01-06

## Оригинальная постановка задачи
Создать сервис для замера и увеличения балла счастья. Mobile-first веб-приложение с будущей конвертацией в мобильное приложение.

## Пользовательские персоны
- **Основная аудитория**: Люди, стремящиеся к саморазвитию и отслеживанию эмоционального состояния
- **Возраст**: 25-45 лет
- **Цель**: Регулярный мониторинг и улучшение психологического благополучия

## Основные требования (Core)
1. **Авторизация**: JWT-based (логин/пароль/подсказка для пароля)
2. **Главная страница**: Спидометр счастья (0-10 баллов) с выбором периода
3. **Страница оценки**: 10 групп гармоничных + 10 групп дисгармоничных состояний
4. **Формула**: Балл счастья = 10 × Σ(гарм) / [Σ(гарм) + Σ(дисгарм)]
5. **Mobile-first дизайн**: Тёмно-фиолетовая тема

## Что реализовано (MVP)
✅ Регистрация/Вход с подсказкой пароля
✅ Восстановление пароля через подсказку
✅ Главная страница со спидометром счастья
✅ Выбор периода (неделя/месяц/квартал/полгода/год/другой)
✅ Страница оценки с 10 группами гармоничных состояний
✅ Страница оценки с 10 группами дисгармоничных состояний
✅ Чекбоксы для индивидуального и группового выбора
✅ Поле рефлексии (саморефлексия)
✅ Выпадающий список решений (Простая задача/Объемная задача/Внедрение привычки)
✅ Модальное окно успешной отправки
✅ Нижняя навигация (5 вкладок)
✅ Расчёт балла счастья по формуле

## Технический стек
- **Backend**: FastAPI + MongoDB
- **Frontend**: React + Tailwind CSS + Shadcn/UI
- **Авторизация**: JWT tokens
- **База данных**: MongoDB
