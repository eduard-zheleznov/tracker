# JoyTracker - Трекер Счастья

## Дата создания: 2026-01-06
## Последнее обновление: 2026-01-06

## Оригинальная постановка задачи
Создать сервис для замера и увеличения балла счастья. Mobile-first веб-приложение с будущей конвертацией в мобильное приложение.

## Пользовательские персоны
- **Основная аудитория**: Люди, стремящиеся к саморазвитию
- **Цель**: Регулярный мониторинг и улучшение психологического благополучия

## Что реализовано (Полная версия) ✅

### Основные разделы

#### 1. Главная страница
- Спидометр счастья (0-10 баллов) с градиентом красный→зелёный
- Выбор периода: 5 кнопок (Неделя/Месяц/Квартал/Пол года/Год) + "Другой период"
- Формула: 10 × Σ(гармоничные) / [Σ(гармоничные) + Σ(дисгармоничные)]

#### 2. Страница Оценка
- 10 групп гармоничных состояний (Группа 1-10)
- 10 групп дисгармоничных состояний
- Чекбоксы для группового и индивидуального выбора
- Поле "Выводы (саморефлексия)" с placeholder текстом
- Выпадающий список решений: Простая задача / Объемная задача / Внедрение привычки
- Кнопка "Отправить" с модальным окном успеха

#### 3. Раздел Анализ (3 подраздела)
- **Повтор состояний**: дисгармоничные (по убыванию), гармоничные (по возрастанию)
- **Тренд силы привычки**: текущее комбо, лучшее комбо, среднее за неделю/месяц
- **Тренд уровня счастья**: баллы по неделям, лучший балл за неделю/месяц

#### 4. Раздел Стратегия (2 подраздела)
- **Решения**: фильтры (#простые/#объемные/#привычки), чекбоксы выполнения, статистика %
- **Рефлексия**: записи из раздела Оценка с датами

#### 5. Раздел Обучение
- Категории с прогрессом (изучено X из Y)
- Видео с отметкой "Изучено"
- Загрузка видео через админ-панель

#### 6. Боковое меню (8 пунктов)
- **Словарь состояний**: поиск, вкладки Гармоничные/Дисгармоничные, описания
- **Вопросы и ответы**: FAQ по категориям + форма задать вопрос
- **Ваш профиль**: О себе, Контакты, Соцсети
- **Напоминания**: 4 типа с временем
- **Ваш психолог**: продающая страница КПТ консультаций
- **Тарифный план**: 3 тарифа (Базовый бесплатно / Продвинутый 299₽ / Премиум 599₽)
- **Слово от автора**: текст о миссии приложения
- **Обратная связь**: рейтинг 5 звёзд + предложения

### Админ-панель (6 вкладок)
- **Обзор**: статистика (пользователи, оценки, вопросы, отзывы)
- **FAQ**: создание, редактирование, публикация вопросов
- **Тексты**: редактирование контента (Психолог, Тарифы, Автор)
- **Категории**: управление категориями обучения
- **Видео**: загрузка и управление видео
- **Отзывы**: просмотр обратной связи
